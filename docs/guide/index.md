# 开始

## 概括

Sptr 是一个用于 Javascript 的模块打包工具，专门用于 ES 模块的打包工具，基于 Rollup 开发，原生支持 Typescript。

## 安装

::: code-group

```bash [npm]
npm i sptr -D
```

```bash [yarn]
yarn add sptr -D
```

```bash [pnpm]
pnpm add sptr -D
```

:::

## 快速开始

安装 Sptr 后在控制台输入 Sptr 就可以打包您的项目了。

```bash
sptr
```

## 兼容性

### 导出 CommonJS

Sptr 默认输出 cjs 文件。
